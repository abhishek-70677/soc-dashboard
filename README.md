# AI-Powered Real-Time SOC Dashboard with Cyber Attack Simulation

## Project Overview

This project is an MCA major project based on Cyber Security and Web Development.

The main goal of this project is to create a modern Security Operations Center (SOC) Dashboard that can:

* Monitor cyber security alerts
* Simulate cyber attacks in real time
* Display attack statistics and security analytics
* Show live alerts and threat activity
* Manage users and reports
* Visualize data using charts and dashboards

The project is being developed using the MERN stack.

# How to Run the Project

## Step 1: Open Project Folder

```bash
cd client
```

## Step 2: Install Dependencies

```bash
npm install
```

## Step 3: Start React Application

```bash
npm start
```

## Step 4: Open Browser

```txt
http://localhost:3000/
```
